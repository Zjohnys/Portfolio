# 🚀 Portfólio - John Lennon Moreira

Um portfólio moderno e responsivo desenvolvido com HTML, CSS e JavaScript puro, apresentando minhas habilidades como desenvolvedor front-end.

## ✨ Características

- **Design Moderno**: Interface limpa e profissional com gradientes e animações suaves
- **Totalmente Responsivo**: Adaptável a todos os dispositivos (desktop, tablet, mobile)
- **Animações Interativas**: Efeitos de scroll, digitação e transições fluidas
- **Performance Otimizada**: Carregamento rápido e experiência de usuário suave
- **Acessibilidade**: Navegação por teclado e estrutura semântica
- **SEO Otimizado**: Meta tags e estrutura HTML adequada

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura semântica e acessível
- **CSS3**: 
  - Flexbox e Grid para layout responsivo
  - Variáveis CSS para consistência de design
  - Animações e transições suaves
  - Media queries para responsividade
- **JavaScript ES6+**:
  - Intersection Observer API para animações
  - Event listeners para interatividade
  - Smooth scroll para navegação
  - Efeitos de partículas no background

## 🚀 Como Executar

1. **Clone o repositório**:
   ```bash
   git clone https://github.com/Zjohnys/portfolio.git
   ```

2. **Navegue até a pasta**:
   ```bash
   cd portfolio
   ```

3. **Abra o arquivo HTML**:
   - Abra `HTML/Main.HTML` em seu navegador
   - Ou use um servidor local como Live Server (VS Code)

## 📱 Responsividade

O portfólio é totalmente responsivo e se adapta aos seguintes breakpoints:

- **Desktop**: > 1104px
- **Tablet**: 991px - 1104px
- **Mobile**: 690px - 991px
- **Mobile Pequeno**: < 690px

## 🔧 Personalização

Para personalizar o portfólio:

1. **Cores**: Edite as variáveis CSS em `:root`
2. **Conteúdo**: Modifique o HTML conforme necessário
3. **Animações**: Ajuste as durações no CSS e JavaScript
4. **Imagens**: Substitua as imagens na pasta `Images/`
